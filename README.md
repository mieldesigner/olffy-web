# Olffy Web

**Papelería ilustrada chilena** — ecommerce web app construida en React + Tailwind CSS v4.

## Stack

- **React 18** + TypeScript
- **Tailwind CSS v4** (via @tailwindcss/vite)
- **Vite** bundler
- **Lucide React** icons
- **Motion** animations

## Páginas

| Sección | Descripción |
|---|---|
| Home | Hero banner, carrusel de productos, galería, footer |
| Tienda | Grid con filtros, detalle de producto, carrito |
| Nuevos Lanzamientos | Colecciones, productos nuevos, próximamente |
| Regalos | Por categoría (amigas, estudiantes, kits, precio, empaque) |
| Nuestra Historia | Historia de marca, proceso, mapa tienda Viña del Mar |
| Contacto | Horarios, mapa Google Maps, redes sociales, FAQ |
| Checkout | Carrito, resumen, códigos promo, extras |
| OLFFY Puntos | Portal cliente (login, dashboard, historial, recompensas, canjes) |
| Admin Panel | Dashboard Shopify, clientes, puntos Supabase, productos, colecciones |

## Tipografía

- **PiepieW01** — displays y headings principales
- **IvyPresto Text Light Italic** — subtítulos editoriales
- **Poppins** — UI funcional
